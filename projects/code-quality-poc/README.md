# Code Quality POC

**Description:** Prototype using Alfred to scan DEX projects and produce actionable code quality improvements.

**Status:** Active

## Objective
Build and validate a repeatable workflow where Alfred:
1. Scans a project repository (starting with TrustVC)
2. Identifies code quality issues (patterns, anti-patterns, coverage gaps, style issues, etc.)
3. Produces a structured report with prioritized recommendations
4. The workflow is templatized for rollout to TradeTrust and OpenCerts
